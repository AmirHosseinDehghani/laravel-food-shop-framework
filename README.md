## 🚀 Features
- Built with Laravel
- Category-based product listing
- Custom RTL layout with animations
- Light/Dark mode switch
- Dynamic footer menu
- Live product search
- Zarinpal sandbox payment integration

---

## 🔧 How to Use

1. **Clone or download** this repository.
2. **Copy** all files to your web directory.
3. **Configure your `.env` file**:
   - Set your DB credentials
   - Set mail, payment, and other services
4. Run these commands:
   ```bash
   composer install
   php artisan migrate --seed
   php artisan serve
