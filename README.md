## ✨ Features
- ✅ Built with Laravel 11
- 🎨 Fully RTL and Persian-friendly layout
- 🌗 Light & Dark Mode with smooth toggle
- 🔍 Live product search with fallback message
- 🧾 Footer menu with animated hover
- 💳 Zarinpal sandbox payment integration
- 🧠 Simple, clean codebase — easy to customize

---

## ⚙️ How to Use

1. **Clone this repo** or download the zip.
2. Copy all files to your server or local project.
3. Set up `.env` file with your info (DB, Zarinpal, etc).
4. Run these:
   ```bash
   composer install
   php artisan migrate --seed
   php artisan serve
